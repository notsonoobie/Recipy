# Recipy Project

`**Work in Progress**` :skull:

## Requirements

You will need to have [Node.js](https://nodejs.org/en/download/) and [Git](git-scm.com/downloads/) installed on your system.

## Installation

    $ git clone :HTTPS-CLONE-ADDRESS
    $ cd Recipy/
    $ npm install

## Developement Mode

    $ npm run dev

## Production Mode
  
    $ npm run build

## Run Client

    $ npm run start

***

## Developer Dependencies

- `Html Webpack Plugin`
- `Webpack`
- `Webpack-cli`
- `Webpack-dev-server`

## Dependencies
